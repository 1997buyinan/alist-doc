### BucKet

存储桶名称

### Endpoint

Endpoint地址

### Region

地区

### Access Key

密钥ID

### Access Secret

访问密钥

### root folder path

根路径，不填默认为根目录。

### Custom Host

自定义cdn加速域名

### url expire time

签名的下载地址有效期，默认为4小时，如果使用自定义加速域名则此项无效。
